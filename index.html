Movie Recommendation System

The Challenge

To build a high-speed, content-based recommendation system that could instantly suggest relevant movies to a user on a mobile app.

My Solution


1. NLP Vectorization: Used NLP (TF-IDF) to convert movie details (plot summary, cast, director, genres) into a numerical feature matrix.
2. Similarity Model: Applied Cosine Similarity to this matrix to calculate a "similarity score" between every movie, allowing for instant lookups.
3. Deployment: Built a lightweight Flask REST API to serve the model. When the app sends a movie title, the API returns a JSON list of the top 10 most similar movies (with posters, cast, etc.) in under a second.
